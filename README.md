# 創英祭2026 | 制作者の愚痴

創英祭2026公式ホームページの裏側にこっそり仕込まれた、制作者（ただの一般生徒 A.N.）による「やけくそボヤキ・アーカイブ」のソースコード管理用リポジトリだて。<br>

<h2> プロジェクト概要 </h2>
このプロジェクトは、創英祭実行委員会からの「納期1週間」「素材・要件定義なし」「適当に作っといて」という究極の無茶振りに応えるため、17歳の一般生徒 (非実行委員) が深夜テンションと意地だけで作り上げた、超個人的なあとがきページらて。
<br>
<h2>イースターエッグの起動仕様</h2>
一般の来場者が誤ってアクセスしないよう、二重のプロテクトロックが施されています。
<br>
<h3>第一段階</h3>
concept.html のメインタイトル最後にあるオレンジ色のピリオド（ . ）を、1秒以内に5回連続でタップする。
<br>
挙動： 成功するとピリオドが赤色に変わり、淡く発光（ロック解除モード移行）。5秒間何も操作しないと自動でリセットされて元のオレンジ色に戻るてば。
<br>
<h3>第二段階</h3>
赤く光っているピリオドを2秒間長押しする。
<br>
挙動： 判定成功で https://ataro-qqq.github.io/guchi/に遷移する。
<br>
<h2>運用上の注意事項</h2>
<h3>実行委員会への秘匿性</h3>
これはあくまで「非公式」の裏ページ。実行委員会や顧問の先生にすぐ見つかると、色々と面倒な調整が入る可能性があるすけ、あまり大っぴらに宣伝しないでくんねぇ。
<br>
<h2>開発者より</h2>
